-LameCode's  AutoTyper-
-======= LITE ========-

This is free of charge auto typer that is compatible with every windows machine that has at least .NET Framework version 4.8 Runtime

!!! If the app crashes right after you open it, it is because you are missing .NET version 4.8. You can download it from official Microsoft website: 

https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48

Look for Runtime and click Download .NET Framework 4.8 Runtime

-====================-
To change the macros, edit files in <EXE folder>\Your Macros.
They are .TXT files, edit them in notepad. Do NOT change file names.
